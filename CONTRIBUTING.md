# Contributing to Candor

Welcome! Project Candor is a systems programming language designed for semantic density and agentic AI collaboration. We are in the early stages (v0.1.0) and welcome all forms of contribution.

## Ways to Contribute

1.  **Bug Reports**: If you find a compiler bug, please open an issue with a minimal `.cnd` reproducer.
2.  **Design Feedback**: Read the [Language Specification](docs/specification.md) and open issues challenging our assumptions about semantic logic and effect tracking.
3.  **Efficiency Benchmarking**: We are actively seeking metrics comparing Candor's token usage against other languages. Help us prove that Candor is more fiscally and energy efficient for AI coding.
4.  **Distributed Examples**: Submit examples that showcase processes spanning networks, emphasizing datacenter scalability.
5.  **AI Collaboration**: Experiment with using AI agents to write Candor code and report on the efficiency of the effect/contract systems.
